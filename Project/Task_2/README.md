# Password Security Checker

This program checks password security by asking the user
to enter randomly selected characters from their password.

Passwords are read from a file provided as a command-line argument.

## How to Run

1. Open a terminal
2. Navigate to the project folder
3. Run:

   python password_checker.py passwords.txt

## Input File Format

One password per line.
Passwords must be at least 9 characters long.

## Features

- Minimum length validation
- Random character verification
- Immediate failure on incorrect input
- File-based input
