This program calculates the total cost of pizza orders using the
Beckett Pizza Plaza 4-for-3 discount offer, where the cheapest pizza
is free.

The program supports multiple orders, delivery or collection,
input validation, and itemized receipts.

## How to Run

1. Open a terminal
2. Navigate to the project folder
3. Run the program:

   python pizza_4for3.py orders.txt

## Input File Format

The input file must contain pizza prices, one per line.
All prices must be positive numbers.

## Features

- 4-for-3 discount
- Multiple orders
- Delivery or collection
- Itemized receipt
- Input validation
